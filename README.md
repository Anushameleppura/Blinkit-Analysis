# 🛒 Blinkit Sales Analysis & Customer Insights

## 📌 Overview

This project analyzes Blinkit sales data to understand customer purchasing behavior, product performance, and outlet characteristics using data analysis and visualization techniques.

---

## 🎯 Objectives

* Analyze customer purchasing patterns
* Identify key factors affecting sales
* Understand impact of outlet size, type, and location
* Generate actionable business insights

---

## 📊 Dataset

* Blinkit sales dataset
* Includes product categories, outlet types, sales, and ratings

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn

---

## 🚀 Project Workflow

1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

---

## 📈 Key Insights

* Certain product categories contribute significantly to total sales
* Outlet size and location influence sales performance
* Customer preferences vary across different outlet types
